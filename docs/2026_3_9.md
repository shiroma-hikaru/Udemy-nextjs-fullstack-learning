## 学んだこと

* shadcn  
shadcnは、React + Tailwind CSSでUIコンポーネントを構築するためのライブラリ。
必要なコンポーネントをCLIで追加して使用する。npmライブラリとしてインストールするのではなく、コードが自分のプロジェクトに直接生成される。

* cn関数  
Tailwind CSS の className を安全に結合するための関数。
条件付きクラスや複数のクラスを整理して記述できる。
    ```
    className={cn(
    buttonVariants({ variant: "secondary" }),
    "px-4"
    )}
    ```
